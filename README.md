# Global Radio — interim CDN content

Real neural-voice narration for the [Global Radio](https://github.com/VivekcMW/GlobalR)
app, generated by `tools/build_catalog.py` (free Microsoft Edge TTS voices).

This is a **free, temporary** hosting location (`raw.githubusercontent.com`)
used until Firebase Storage billing is enabled on the real project — see
`global_radio/tools/FUTURE_FIREBASE_SETUP.md` in the main repo for the
migration plan. Not meant as a permanent production CDN (no cache headers,
no custom domain, subject to GitHub's raw-content rate limits).

## Layout

```
catalog.json
{language}/{voiceId}/{itemId}.mp3
```

Matches the app's `audioUrlFor` convention exactly (`lib/data/models/catalog_item.dart`):
`{CDN_BASE}/{language}/{voiceId}/{itemId}.mp3`.

## Regenerating

From the main repo:

```bash
cd global_radio
python tools/build_catalog.py
# copy tools/../cdn_dist/* into this repo, commit, push
```
